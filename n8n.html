Что такое n8n?

n8n — open-source альтернатива Zapier и Make. Позволяет соединять любые сервисы без программирования: Email, AI, CRM, Telegram, базы данных, платежи и многое другое. Бесплатно для самостоятельного использования.

FAQ по n8n

Частые вопросы про n8n

Нужен ли свой сервер для n8n?

Да, для использования шаблонов нужен собственный n8n (self-hosted или облачный). Базовая установка на VPS занимает 5 минут через Docker. В каталоге есть инструкция.

Как импортировать шаблон?

В боте нажми «Скопировать JSON» → в своём n8n открой Workflows → Import from Clipboard. Всё. Шаблон готов к использованию.

Какие API ключи нужны?

Зависит от шаблона. Для AI-шаблонов нужен API ключ OpenAI/Claude. Для Telegram — токен бота. Для Gmail — OAuth. В каждом шаблоне указаны необходимые credentials.

Сколько стоит подписка?

100 ⭐ в месяц. В каждой категории — 3 бесплатных шаблона для теста. Подробнее на странице Цены и тарифы.
