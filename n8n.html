FAQ

Что такое n8n?

n8n — open-source альтернатива Zapier и Make. Соединяет сервисы без программирования. Бесплатно для самостоятельного использования.

Как использовать шаблоны?

1. Открой @Rob46_bot 2. Выбери категорию и шаблон 3. Скачай JSON 4. Импортируй в n8n: Workflows → Import from File

Есть ли бесплатный доступ?

Да. В каждой категории 3 шаблона бесплатно. Полный каталог — ⭐ 100/мес.
