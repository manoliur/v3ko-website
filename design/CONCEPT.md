# V3Ko — AI-продукты и автоматизации

## Концепция
V3Ko — платформа AI-продуктов. Каждый продукт решает конкретную задачу через AI и автоматизацию.
Все продукты доступны через Telegram-ботов с оплатой звёздами.

## Дизайн
- **Тема:** Светлая, тёплая, дружелюбная — рассчитана на новичков
- **Цвета:** Кремовый фон (#FFFBF6), коралл (#FF6B57), мятный акцент (#14B8A6), мягкий фиолетовый (#7C6BE8), золотой для звёзд (#FFB627)
- **Стиль:** Крупные скругления, мягкие тёплые тени, пилюли-кнопки, много воздуха
- **Тон текста:** Простыми словами, без жаргона — объясняем термины (n8n, промпты, Stars)
- **Конверсия:** Каждая карточка продукта → CTA в Telegram-бота; на главной блок «Как начать за 3 шага»

## Продукты
1. 🔧 **N8N Каталог** — 9 580 готовых шаблонов автоматизации (@Rob46_bot)
2. 📝 **Промпты дня** — Ежедневные промпты для нейросетей (закрытый канал)
3. 🎙 **Voice Assistant** — Голосовой AI-ассистент на русском (@VoiceV3k_bot)
4. 🤖 **AI-редакция** — Автоматическое создание контента (в разработке)
