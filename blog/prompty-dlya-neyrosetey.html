Промпты для нейросетей: как писать эффективные запросы

5 мин чтения · 6 июля 2026 · промпты AI

Промпт (prompt) — это запрос, который вы отправляете нейросети. Качество ответа напрямую зависит от того, как составлен промпт. Хороший промпт — это чёткая инструкция, которая не оставляет пространства для двусмысленностей.

Универсальная формула промпта

Для любой нейросети работает одна и та же структура:

[Роль] + [Контекст] + [Задача] + [Формат] + [Ограничения]

Пример плохого промпта: «Напиши пост для Instagram»

Пример хорошего промпта: «Ты — SMM-менеджер (роль). Мы запускаем AI-сервис для автоматизации (контекст). Напиши пост-знакомство на 500 знаков (задача). Используй дружеский тон, добавь 3 хештега (формат). Без эмодзи и восклицательных знаков (ограничения)»

Техники промптов для ChatGPT

1. Chain-of-Thought (Цепочка рассуждений)

Попросите AI рассуждать шаг за шагом. Вместо «Реши задачу» — «Реши задачу по шагам, объясняя каждый шаг». Это dramatically улучшает точность для сложных задач.

2. Few-Shot (Примеры в промпте)

Дайте 2-3 примера того, что хотите получить. ChatGPT отлично улавливает паттерны:

«Напиши 3 варианта приветствия для email-рассылки. Формат как в примере:
Пример: "Привет, {имя}! 👋 Готовы к переменам?" → "Привет, {имя}! 🚀 Ваш чек-лист готов."»

3. Ролевой промптинг (System Prompt)

Чётко задайте роль. ChatGPT работает лучше, когда знает контекст:

  • «Ты — эксперт по маркетингу с 10-летним опытом»
  • «Ты — Python-разработчик уровня Senior»
  • «Ты — копирайтер, специализирующийся на продающих текстах»

Промпты для Midjourney

Для Midjourney важны визуальные описания и параметры. Базовая структура:

[Объект] + [Детали] + [Окружение] + [Освещение] + [Стиль] + [Параметры]

Пример: Viking longship emerging from thick fog, photorealistic, dramatic lighting, cinematic, 8k --ar 16:9 --v 6

Ключевые параметры Midjourney:

  • --ar 16:9 — соотношение сторон (широкий формат)
  • --v 6 — версия модели (сейчас актуальна v6)
  • --s 250 — стилизация (50-1000, чем выше — тем артистичнее)
  • --iw 2 — вес изображения (при работе с reference)

Промпты для Claude

Claude отлично работает с кодом и техническими задачами. Лучшие практики:

  • Один запрос — одна задача: Claude лучше фокусируется на конкретной проблеме
  • Прикрепляйте контекст: Дайте код, документацию или описание архитектуры
  • Используйте XML-теги: Claude понимает структурированные запросы

«Review this PR. Check: logic errors, edge cases, performance issues, security vulnerabilities, code style. Here's the diff: ...»

Промпты для Stable Diffusion

SD требует более технических описаний. Важно указывать:

  • Negative prompt: что НЕ должно быть на картинке
  • Sampler и Steps: DPM++ 2M Karras, 20-30 steps
  • CFG Scale: 7-9 для баланса следования промпту

Пример: Majestic mountain peak at golden hour, clouds swirling around summit, alpine lake reflection, photorealistic С negative prompt: blurry, low quality, deformed, watermark

Чек-лист хорошего промпта

  1. ✅ Определена роль AI
  2. ✅ Дан контекст / background
  3. ✅ Чётко сформулирована задача
  4. ✅ Указан формат ответа
  5. ✅ Добавлены ограничения (длина, тон, стиль)
  6. ✅ Использованы примеры (few-shot) для сложных задач
  7. ✅ Запрос на русском или английском (в зависимости от модели)

Типичные ошибки при составлении промптов

1. Слишком размытая формулировка

«Напиши текст» — плохо. «Напиши пост для Telegram на 300 знаков о нашем новом AI-сервисе, используй энергичный тон и добавь 3 преимущества» — хорошо. Чем конкретнее запрос, тем точнее ответ нейросети.

2. Отсутствие контекста

Нейросеть не знает ваш бизнес, если вы ей не рассказали. Всегда добавляйте: кто вы, для кого пишете, какая у вас целевая аудитория, какой тон общения принят в компании.

3. Слишком много задач в одном промпте

«Напиши 10 постов, сделай анализ конкурентов, составь стратегию и нарисуй логотип» — AI сделает всё, но поверхностно. Разбивайте сложные задачи на последовательные простые промпты.

4. Игнорирование формата вывода

Если не указать формат, AI может ответить в виде эссе, когда вам нужен список. Всегда указывайте: «в виде таблицы», «списком из 5 пунктов», «в формате JSON», «не более 200 слов».

Промпты для бизнес-задач

Для отдела продаж

Промпт: «Ты — опытный менеджер по продажам. Напиши 3 скрипта холодного звонка для разных типов клиентов: 1) владелец малого бизнеса, 2) руководитель отдела, 3) директор по маркетингу. В каждом скрипте: приветствие, выявление боли, предложение ценности, призыв к действию. Тон — уверенный, без навязчивости.»

Для поддержки клиентов

Промпт: «Ты — специалист техподдержки IT-компании. Ответь клиенту, который пишет: «У меня не работает оплата на сайте». Ответ должен содержать: 1) эмпатию и понимание, 2) 3 самых частых причины проблемы, 3) пошаговую инструкцию решения, 4) контакты для экстренной связи. Тон — дружеский, профессиональный.»

Для контент-мейкеров

Промпт: «Ты — редактор Telegram-канала про AI. Напиши пост о нейросетях для начинающих. Формат: заголовок-вопрос, 3 факта, вывод-призыв подписаться. Без сложных терминов. 250-300 знаков. Добавь 3 релевантных хештега.»

Шаблоны промптов для быстрого старта

Скопируй и подставь свои данные:

Шаблон для анализа текста

«Проанализируй следующий текст: [текст]. Выдели: главную мысль, 3 ключевых аргумента, слабые места, рекомендации по улучшению. Ответ в виде таблицы.»

Шаблон для генерации идей

«Ты — [роль]. Мне нужно [цель]. Предложи 10 идей в формате: 1) Название, 2) Краткое описание на 1 предложение, 3) Почему это сработает. Без абстрактных советов, только конкретика.»

Шаблон для перевода и адаптации

«Переведи следующий текст с [язык] на [язык]. Сохрани тон и стиль оригинала. Адаптируй культурные отсылки под [регион]. Выдели цветом термины, которые требуют уточнения.»

Часто задаваемые вопросы

Какой длины должен быть промпт?

Золотая середина — 2-5 предложений. Слишком короткий промпт даёт общий ответ. Слишком длинный — AI может упустить важные детали. 100-300 слов — оптимально для большинства задач.

На каком языке писать промпты?

ChatGPT и Claude отлично работают с русским языком. Midjourney лучше понимает английский — пишите описания на английском для лучшего качества картинок. Stable Diffusion одинаково хорошо работает с обоими языками.

Можно ли использовать один промпт несколько раз?

Да, но лучше адаптировать под конкретную задачу. Промпт, который сработал для поста в Instagram, не подойдёт для email-рассылки. Сохраняйте удачные промпты как шаблоны и меняйте только переменную часть: тему, продукт, аудиторию.
